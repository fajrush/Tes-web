# Sistem Pembayaran Otomatis & Terintegrasi WhatsApp (Vercel Ready)

Proyek arsitektur backend + panel admin yang didesain ramah cloud serverless.

## 🚀 Langkah Deploy:
1. Push seluruh file ini ke akun GitHub lu.
2. Hubungkan repository ke Vercel.
3. Berikan Environment Variables `ADMIN_PASSWORD` dan `WHATSAPP_GATEWAY_URL` di halaman Vercel.
4. Buka admin panel web lu di: `https://nama-web-lu.vercel.app/admin.html`
